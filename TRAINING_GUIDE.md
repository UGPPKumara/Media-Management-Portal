# Module 2: Creator Training

## Lesson 2.1: Understanding Your Dashboard

As a Creator, your dashboard shows:

| Card | Description |
|------|-------------|
| **Total Posts** | All posts you've created |
| **Drafts** | Saved but not submitted |
| **Pending** | Waiting for review |
| **Approved** | Ready for publishing |
| **Rejected** | Need revision |

---

## Lesson 2.2: Creating a New Post

### Step-by-Step

1. Click "Create Post" in sidebar
2. Fill in the form:

| Field | Required | Description |
|-------|----------|-------------|
| Title | Yes | Clear, descriptive title |
| Description | No | Additional details |
| Media | Yes | Image or video file |

3. Upload your media file:
   - Click "Choose File"
   - Select your image/video
   - Wait for preview to appear

4. Choose action:
   - **Save as Draft** ‚Üí Save for later
   - **Submit for Review** ‚Üí Send to manager

### Best Practices

‚úÖ Use descriptive titles  
‚úÖ Add context in description  
‚úÖ Check file size before upload (max 50MB)  
‚úÖ Preview before submitting

---

## Lesson 2.3: Managing Your Posts

### Viewing Posts

1. Click "My Posts" in sidebar
2. View all your posts in a list/grid

### Post Status Colors

| Color | Status | Meaning |
|-------|--------|---------|
| üîµ Blue | DRAFT | Not submitted |
| üü° Yellow | PENDING | Awaiting review |
| üü¢ Green | APPROVED | Ready to publish |
| üî¥ Red | REJECTED | Needs changes |
| üü£ Purple | PUBLISHED | Live on social |

### Editing a Post

> ‚ÑπÔ∏è You can only edit DRAFT or REJECTED posts

1. Find your post
2. Click "Edit" button
3. Make changes
4. Save or submit

### Deleting a Post

1. Find your post
2. Click "Delete" button
3. Confirm deletion

> ‚ö†Ô∏è Deleted posts cannot be recovered!

---

## Lesson 2.4: Handling Rejections

When a post is rejected:

1. Go to your posts list
2. Find the rejected post (red status)
3. Click to view rejection reason
4. Click "Edit" to fix issues
5. Submit again for review

**Example rejection reasons:**

- "Image quality too low"
- "Content doesn't match guidelines"
- "Missing required information"

# Module 3: Manager Training

## Lesson 3.1: Manager Dashboard

As a Manager, you see additional information:

| Section | Description |
|---------|-------------|
| System Stats | All posts, all users |
| Pending Review | Posts waiting for you |
| Recent Activity | Latest submissions |

---

## Lesson 3.2: Reviewing Posts

### Finding Pending Posts

1. Go to "Posts" page
2. Filter by "Pending" status
3. See all posts awaiting review

### Reviewing a Post

1. Click on a pending post
2. View the title and description
3. Preview the media content
4. Decide: Approve or Reject

---

## Lesson 3.3: Approving Content

### Steps to Approve

1. Open the pending post
2. Review content quality
3. Click "Approve" button
4. Confirm approval
5. Post status changes to APPROVED

### Approval Checklist

Before approving, verify:

- [ ] Content matches guidelines
- [ ] Image/video quality is acceptable
- [ ] Title is appropriate
- [ ] No offensive content
- [ ] Factually accurate

---

## Lesson 3.4: Rejecting Content

### Steps to Reject

1. Open the pending post
2. Click "Reject" button
3. Enter rejection reason (REQUIRED)
4. Click "Submit Rejection"

### Writing Good Rejection Feedback

| ‚ùå Bad Example | ‚úÖ Good Example |
|----------------|-----------------|
| "Bad" | "Image resolution too low. Please upload at least 1080p." |
| "Wrong" | "Content doesn't match brand guidelines. See policy doc." |
| "Fix it" | "Please add a more descriptive caption explaining the product." |

---

## Lesson 3.5: Publishing Content

### Publishing Approved Posts

1. Find approved post
2. Click "Publish" button
3. Select platform (Facebook)
4. Confirm publishing
5. Status changes to PUBLISHED

## Lesson 4.2: Managing Users

### Viewing Users

1. Go to "Users" page
2. View all registered users

### Creating a New User

1. Click "Add User"
2. Fill in:
   - Username
   - Email
   - Password
   - Role (Creator/Manager/Admin)
3. Click "Create"

### Changing User Roles

1. Find user in list
2. Click role dropdown
3. Select new role
4. Confirm change

### Deactivating Users

1. Find user in list
2. Toggle status switch
3. User cannot login when deactivated

## Lesson 4.3: System Settings

### Updating Company Branding

1. Go to "Settings"

   - Company name
   - Company logo
